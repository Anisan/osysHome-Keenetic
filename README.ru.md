# Keenetic - Интеграция с роутерами Keenetic

![Keenetic Icon](static/Keenetic.png)

Интеграция с роутерами Keenetic для мониторинга подключенных устройств и управления сетью.

## Описание

Модуль `Keenetic` предоставляет интеграцию с роутерами Keenetic для платформы osysHome. Он обеспечивает мониторинг подключенных устройств, статистику сети и управление устройствами.

## Основные возможности

- ✅ **Мониторинг устройств**: Мониторинг устройств, подключенных к роутеру Keenetic
- ✅ **Статистика сети**: Просмотр статистики использования сети
- ✅ **Управление устройствами**: Управление устройствами роутера
- ✅ **Несколько роутеров**: Поддержка нескольких роутеров Keenetic
- ✅ **Интеграция поиска**: Поиск устройств и роутеров
- ✅ **Поддержка виджетов**: Виджет панели управления со статистикой роутера

## Панель администратора

Модуль предоставляет административный интерфейс для:
- Просмотра роутеров
- Управления конфигурацией роутера
- Просмотра подключенных устройств
- Настройки параметров устройств

## Конфигурация

- **Настройки роутера**: Хост, порт, имя пользователя, пароль
- **Интервал обновления**: Интервал обновления устройств (по умолчанию: 5 секунд)
- **Конфигурация устройств**: Связывание устройств с объектами osysHome

## Использование

### Добавление роутера

1. Перейдите в модуль Keenetic
2. Нажмите "Add Router"
3. Введите учетные данные роутера
4. Сохраните конфигурацию
5. Устройства обнаруживаются автоматически

## Технические детали

- **API**: HTTP API Keenetic
- **Многопоточность**: Параллельная обработка роутеров
- **Обнаружение устройств**: Автоматическое обнаружение устройств
- **Связывание свойств**: Связывание устройств роутера с объектами
